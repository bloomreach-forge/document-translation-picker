Hippo CMS - Document Translation Picker

The document translation picker project provides a plugin to Hippo CMS. It enables editors to pick existing documents as a translation for a document.

Usage

This plugin adds a submenu item to the 'document' item in the toolbar. A CMS user can use it to open a dialog and select existing documents in translated folders. These documents will get the same, newly created, translationID as the current document.
